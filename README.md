# 🎂 妈妈生日快乐 - 生日祝福网页

一个唯美梦幻的生日祝福网页，为妈妈的生日增添特别的魔力✨

## ✨ 功能特性

### 🏠 首页面
- 渐变色"妈妈生日快乐"标题
- 点击开始按钮进入游戏
- 浮动心形动画

### 🎁 游戏页面 - 盲盒抽奖
- 九宫格抽奖系统
- 中心大奖（心意许愿卷🎁）
- 固定抽奖顺序：
  1. 玩具🧸
  2. 花朵💐
  3. 谢谢惠顾😢
  4. 转账520💰
  5. 再循环...
  6. 最终：心意许愿卷🎁
- 点击任意格子开始抽奖
- 抽到中心大奖自动跳转祝福页面

### 💬 祝福页面
- 聊天冒泡形式的祝福语
- 16条温馨生日祝福
- 左右交替显示效果
- 两侧唯美烟火特效

## 🎯 使用说明

1. **首页面**：点击"点击开始"按钮进入游戏
2. **游戏页面**：点击9个格子中的任意一个进行抽奖
3. **继续游戏**：可以继续点击其他未打开的格子
4. **获得大奖**：当抽到中心大奖"心意许愿卷🎁"时，自动跳转到祝福页面
5. **祝福放送**：享受温馨的生日祝福和梦幻烟火特效

## 💡 自定义说明

### 修改祝福语
在 `script.js` 中找到 `blessingMessages` 数组：

```javascript
const blessingMessages = [
    '您的祝福语1',
    '您的祝福语2',
    // ...更多祝福
];
```

### 修改颜色主题
在 `style.css` 中修改渐变色和RGB值：
- 主色调：`#ff69b4` (粉色)
- 辅色调：`#ffd700` (金色)
- 背景：`#0f0c29` (深紫色)

### 修改奖励顺序
在 `script.js` 中的 `gameState.prizeOrder` 数组中修改
